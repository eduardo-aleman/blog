Useful RPC Commands for a Nano Node

Useful RPC Commands for a Nano Node Once your Nano node is running and port 7076 is open, you can interact with it using HTTP POST requests to the JSON-RPC API. Below is a collection of useful commands organized by category. 🔍 Node Status curl -d '{ "action": "version" }' -H "Content-Type: application/json" http://127.0.0.1:7076 curl -d '{ "action": "telemetry" }' -H "Content-Type: application/json" http://127.0.0.1:7076 curl -d '{ "action": "block_count" }' -H "Content-Type: application/json" http://127.0.0.1:7076 🧭 Network & Peers curl -d '{ "action": "peers" }' -H "Content-Type: application/json" http://127.0.0.1:7076 curl -d '{ "action": "peer_add", "address": "peering.nano.org", "port": "7075" }' \ -H "Content-Type: application/json" http://127.0.0.1:7076 🔐 Wallets & Accounts curl -d '{ "action": "wallet_list" }' -H "Content-Type: application/json" http://127.0.0.1:7076 curl -d '{ "action": "wallet_create" }' -H "Content-Type: application/json" http://127.0.0.1:7076 curl -d '{ "action": "account_create", "wallet": "<WALLET_ID>" }' \ -H "Content-Type: application/json" http://127.0.0.1:7076 curl -d '{ "action": "wallet_add", "wallet": "<WALLET_ID>", "key": "<PRIVATE_KEY>" }' \ -H "Content-Type: application/json" http://127.0.0.1:7076 💬 Account Information curl -d '{ "action": "account_balance", "account": "<ACCOUNT_ID>" }' \ -H "Content-Type: application/json" http://127.0.0.1:7076 curl -d '{ "action": "account_info", "account": "<ACCOUNT_ID>" }' \ -H "Content-Type: application/json" http://127.0.0.1:7076 🛠 Administration curl -d '{ "action": "representatives_online" }' \ -H "Content-Type: application/json" http://127.0.0.1:7076 curl -d '{ "action": "bootstrap_any" }' \ -H "Content-Type: application/json" http://127.0.0.1:7076 🗂 References Official Nano RPC Documentation NanoLooker Explorer Nano Node Directory ⚠️ Disclaimer This guide is intended to document practical JSON-RPC commands for interacting with a self-hosted Nano node.
